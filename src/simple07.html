Using INPUT

Another way to give the computer information is with the command INPUT. INPUT lets you put information into the computerâ€™s memory while the program is running. This means that you can give the computer different information, each time you run a program.

You always put a variable name after INPUT. When the computer meets INPUT it waits for you to type something and stores it in the named variable.

This computer tells you it is waiting by displaying a question mark.

10 CLS
20 PRINT "WHAT IS YOUR NAME?"
30 INPUT N$
40 PRINT "HOW OLD ARE YOU?"
50 INPUT A
60 CLS
70 PRINT N$; " IS "; A
80 PRINT "YEARS OLD"
The line INPUT N$ tells the computer to expect words because the variable name ends with $, while the line INPUT A tells the computer to expect a number.
