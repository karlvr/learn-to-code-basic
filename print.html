Welcome

On the righthand side of the screen is your computer. You can only interact with this computer by typing as it doesn’t use a mouse.

On this page we’re going to learn how to enter programs into this computer.

Need Input

We’re going to type in your first program! Type in the following lines, including the line numbers, pressing [return] after typing each line:

10 PRINT "What is your name
20 INPUT a$
30 PRINT "Nice to meet you " a$

If the computer says Syntax error or anything strange happens, check that you’ve typed exactly what appears above including the line numbers.

Each line starts with its line number. If you make a mistake on a line you can edit it using its line number. Type EDIT 10 to edit line 10, then press [return] to finish editing.

When you’ve finished typing your program type LIST to show your program. If you’ve made any mistakes, edit the line that contains the mistake. If you’ve added extra lines you can remove them using the line number, for example DELETE 10.

Run your program by typing RUN.

Print

The PRINT command prints whatever comes after it. On line 10 we gave PRINT a string literal, that is a string of text surrounded by ""s. You could type whatever you like inside those quotes. Type EDIT 10 now to change the message that gets printed out. Type RUN to run your new program.

If you type a line that doesn’t start with a line number, the computer runs that command immediately rather than adding it to your program to run later. Let’s try printing a few different things to see how versatile the PRINT command is.

PRINT 42
PRINT "Basic"
PRINT 3.14
PRINT 8*7

The last line above multiplies two numbers. Try printing a few more equations. You can use * for multiply, + and - as usual, and / for divide. You can also use () for more complex equations.

Uppercase lowercase

This programming language is called BASIC. It doesn’t matter whether you use uppercase letters or lowercase letters to type its commands, like PRINT, or variables, like a$; BASIC treats uppercase, lowercase or mixed-case as the same. Where it does matter is in strings.

print "It matters in strings
input A$
print "But not in variables
print a$
